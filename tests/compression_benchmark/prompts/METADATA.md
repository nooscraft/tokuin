# Compression Benchmark Dataset

This directory contains real-world prompts collected from public sources for testing prompt compression functionality.

## Dataset Information

- **Total Prompts**: 75
- **Categories**: instructions, structured, conversations, technical, mixed
- **Collection Date**: 2025-12-09

## Categories

### Instructions
Pure instruction prompts

**Count**: 39

### Structured

**Count**: 16

**Count**: 6

**Count**: 7

**Count**: 7
